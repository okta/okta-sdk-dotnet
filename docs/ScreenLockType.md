# Okta.Sdk.Model.ScreenLockType
Indicates the type of screen lock required for device assurance. **Note:** When used in `screenLockType.include`, only certain combinations are valid: - `[BIOMETRIC]` - Requires biometric screen lock - `[PASSCODE, BIOMETRIC]` - Requires passcode or biometric screen lock The value `NONE` and single `[PASSCODE]` are not valid for device assurance policies.
